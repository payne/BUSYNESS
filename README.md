# Development

Install

    npm install

Configure

* Rename `twitter_config.sample.json` to `twitter_config.json` and add your own twitter keys obtained from [dev.twitter.com](https://dev.twitter.com/).

Run

    node app.js

Browse

    http://localhost:3000

# Production

Heroku Environment Variables

* `NODE_ENV = 'production'`
* `consumerKey = ''`
* `consumerSecret = ''`
